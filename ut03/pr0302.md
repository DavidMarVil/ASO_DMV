Ejercicio 1: Menú de operaciones matemáticas
Crea un script que presente un menú con las operaciones básicas (suma, resta, multiplicación, división) y solicite al usuario que elija una operación. El script debe ejecutar la operación seleccionada utilizando case.

````
echo " MENÚ DE OPERACIONES  "
echo "1) Suma"
echo "2) Resta"
echo "3) Multiplicación"
echo "4) División"
echo "5) Salir"

read -p "Elige una opción (1-5): " opcion


if [ "$opcion" -eq 5 ]; then
    echo "Saliendo del programa..."
    exit 0
fi

read -p "Introduce el primer número: " num1
read -p "Introduce el segundo número: " num2

case $opcion in
    1)
        resultado=$((num1 + num2))
        echo "La suma es: $resultado"
        ;;
    2)
        resultado=$((num1 - num2))
        echo "La resta es: $resultado"
        ;;
    3)
        resultado=$((num1 * num2))
        echo "La multiplicación es: $resultado"
        ;;
    4)
        # Comprobamos que el divisor no sea cero
        if [ "$num2" -eq 0 ]; then
            echo "Error: no se puede dividir entre 0"
        else
            resultado=$((num1 / num2))
            echo "La división es: $resultado"
        fi
        ;;
    *)
        echo "Opción no válida"
        ;;
esac
````
Ejercicio 2: Identificación de extensión de archivo
Haz un script que solicite al usuario un nombre de archivo y, usando case, determine si es un archivo de texto (.txt), un archivo comprimido (.zip o .tar.gz), o cualquier otro tipo de archivo
````
#!/bin/bash
# Ejercicio 2: Identificación de extensión de archivo

read -p "Introduce el nombre del archivo: " archivo

case "$archivo" in
    *.txt)
        echo "El archivo es un archivo de texto (.txt)"
        ;;
    *.zip)
        echo "El archivo es un archivo comprimido (.zip)"
        ;;
    *.tar.gz)
        echo "El archivo es un archivo comprimido (.tar.gz)"
        ;;
    *)
        echo "Tipo de archivo desconocido o no reconocido"
        ;;
esac

````
Ejercicio 3: Conversor de unidades
Crea un script que ofrezca un menú para convertir entre diferentes unidades de longitud (metros a kilómetros, pies a metros, etc.) utilizando case para gestionar las conversiones
````
#!/bin/bash
# Ejercicio 3: Conversor de unidades (versión básica)

echo " CONVERSOR DE UNIDADES "
echo "1) Metros a Kilómetros"
echo "2) Kilómetros a Metros"
echo "3) Pies a Metros"
echo "4) Metros a Pies"

read -p "Elige una opción (1-4): " opcion

read -p "Introduce el valor a convertir: " valor

case $opcion in
    1)
        resultado=$((valor / 1000))
        echo "$valor metros son $resultado kilómetros."
        ;;
    2)
        resultado=$((valor * 1000))
        echo "$valor kilómetros son $resultado metros."
        ;;
    3)
        # 1 pie = 0.3048 metros (usamos aproximación entera)
        resultado=$((valor * 3 / 10))
        echo "$valor pies son aproximadamente $resultado metros."
        ;;
    4)
        # 1 metro = 3.281 pies (aproximación)
        resultado=$((valor * 3))
        echo "$valor metros son aproximadamente $resultado pies."
        ;;
    *)
        echo "Opción no válida."
        ;;
esac

````

Ejercicio 4: Menú de configuración del sistema
Realiza un script que presente un menú con varias opciones de configuración del sistema: por ejemplo, apagar, reiniciar o cerrar sesión. Usa case para gestionar las opciones seleccionadas.
````
#!/bin/bash

echo "1) Apagar"
echo "2) Reiniciar"
echo "3) Cerrar sesión"
echo "4) Salir"
read -p "Elige una opción: " opcion

case $opcion in
    1)
        echo "Apagando el sistema..."
        sudo shutdown -h now
        ;;
    2)
        echo "Reiniciando el sistema..."
        sudo reboot
        ;;
    3)
        echo "Cerrando sesión..."
        gnome-session-quit --logout --no-prompt
        ;;
    4)
        echo "Saliendo..."
        ;;
    *)
        echo "Opción no válida."
        ;;
esac
````

Ejercicio 6: Clasificación de notas
Realiza un script que solicite una calificación numérica y, usando case, la clasifique como “Sobresaliente”, “Notable”, “Aprobado”, o “Suspenso” según el rango de valores.
````
#!/bin/bash

read -p "Introduce una calificación (0-10): " nota

case $nota in
    10|9)
        echo "Sobresaliente"
        ;;
    8|7)
        echo "Notable"
        ;;
    6|5)
        echo "Aprobado"
        ;;
    4|3|2|1|0)
        echo "Suspenso"
        ;;
    *)
        echo "Calificación no válida. Debe estar entre 0 y 10."
        ;;
esac

````

Ejercicio 8: Control de servicios en Linux
Crea un script que pregunte por el nombre de un servicio y luego presente un menú para iniciar, detener o reiniciar un servicio en Linux (como apache2 o nginx). Usa case para gestionar las opciones y los comandos correspondientes (systemctl start, stop, restart)
````
#!/bin/bash


read -p "Introduce el nombre del servicio (ej. apache2, nginx): " servicio

echo " MENÚ DE CONTROL DEL SERVICIO "
echo "1) Iniciar el servicio"
echo "2) Detener el servicio"
echo "3) Reiniciar el servicio"
echo "4) Salir"
read -p "Elige una opción (1-4): " opcion

case $opcion in
    1)
        echo "Iniciando el servicio $servicio..."
        sudo systemctl start $servicio
        ;;
    2)
        echo "Deteniendo el servicio $servicio..."
        sudo systemctl stop $servicio
        ;;
    3)
        echo "Reiniciando el servicio $servicio..."
        sudo systemctl restart $servicio
        ;;
    4)
        echo "Saliendo..."
        ;;
    *)
        echo "Opción no válida."
        ;;
esac

````
